## HR attrition analysis dashboard

## Projekt célja
Power BI-ban analizálni egy cég lemorzsolódási adatait és egy dashboard segítségével vizualizálni a lekövetkeztetett összefüggéseket

## Dashboard oldal
- Alkalmazottak száma
- Lemorzsolódások száma, aránya
- Átlagos havi bevitel
- Magas kockázatú alkalmazottak
- Túlórában dolgozó lemorzsolódott alkalmazottak aránya
- Elégedettség szint szerinti lemorzsolódások száma
- Túlórában és rendes munkaidőben dolgozó lemorzsolódottak összehasonlítása
- Átlagos havi kereset életkor szerint

## Fő insightok
- Túlórázó dolgozók nagyobb arányban hagyják el a céget
- Elégedettséggel nincs jelentős összefüggésben a fluktuáltak száma
- Nagy mértékben eltér a fluktuációk száma a cégen belüli osztályokon
- DAX mérés segítségével jól elkülöníthetők a kockázatos dolgozók
