## HR attrition analysis dashboard

## Project goal
Analyze attrition statistics in a company using Power BI dashboard

## Dashboard page
- Total employees
- Attritions number and rate
- Average monthy income
- Number of high risk employees
- Rate of attritions among employees working overtime
- Number of attritions by job satisfaction
- Comparison between attritted employees who work regular hours versus overtime
- Average monthly income by age

## Main insights
- Employees who work overtime are more likely to attrit
- The number of attritions aren't related closely with job satisfaction
- There is a large disparity of attritions between the different departments
- We can isolate the high risk employees from the others well using DAX measures
